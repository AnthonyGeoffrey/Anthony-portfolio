# 🌐 Anthony Geoffrey Portfolio

A modern, responsive personal portfolio website built to showcase my projects, skills, and journey as a student developer.

🔗 Live Demo: https://anthony-portfolio-mu.vercel.app/

---

## 📌 About The Project

This portfolio was created to serve as a central hub for my work, allowing recruiters, collaborators, and peers to explore my skills and projects in one place.

It highlights:
- My development skills
- Projects I’ve built
- Contact information for opportunities

---

## 🚀 Features

- ⚡ Fully responsive design (mobile + desktop)
- 🎨 Modern UI with smooth animations
- 🧩 Projects showcase section
- 📞 Contact section for easy communication
- 🌙 Dark themed futuristic design

---

## 🛠️ Built With

- **React (Vite)**
- **Tailwind CSS**
- **Framer Motion**
- **Vercel (Deployment)**
